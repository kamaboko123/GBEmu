# Gameboy Emulator
学習用に開発を進めているゲームボーイエミュレータです。  

## ビルド環境
- Windows10
- VisualStudio Community 2022

- Dependent libraries
    - [SDL2](https://www.libsdl.org/)
    - [imgui](https://github.com/ocornut/imgui)
